QLearningGame

Author: Jose, Anand

File descriptions:
  agents.py - class definitions for table-based Q-learning bot, and deep Q-learning bot
  config.py - a settings file (currently only used to set the action space and grid size)
  deepQ.py - functions related to the deep Q-learning bot (currently contains a function that can be used to create the neural network)
  games.py - class definitions for various games (currently contains only one; more can be added but minor changes would have to be made in other files for compatibility)
  main.py - the main file from where everything else is called
  plots.py - this file contains helper functions that can be used to visualise things
  tests.py - this file contains a test function
  train.py - this file contains a function used to train any bot, for any game (very general!)
